## BUILD GUARDRAILS WAJIB

### A. Guardrails Performa
- gunakan HTML semantik
- minimalkan nested div yang tidak perlu
- Bootstrap hanya untuk komponen yang dipakai
- hindari plugin JS berlebih
- hindari library tambahan tanpa alasan kuat
- semua gambar dioptimalkan
- font dibatasi
- script non-kritis di-defer
- third-party script dibatasi
- hero harus ringan
- layout shift harus dicegah
- no decorative bloat

### B. Guardrails Struktur
- penamaan file harus konsisten
- struktur folder harus stabil
- asset harus rapi
- semua halaman baru harus mengikuti template
- partial/include harus konsisten bila dipakai
- metadata dan schema tidak boleh terlewat

### C. Guardrails SEO
- 1 halaman = 1 intent utama
- 1 halaman = 1 keyword cluster utama
- metadata harus unik
- URL/filename harus bersih
- heading harus hierarkis
- canonical harus jelas
- schema harus relevan

### D. Guardrails AEO/GEO
- halaman money page wajib punya jawaban cepat
- wajib ada FAQ relevan
- trust signal wajib tampil
- identitas brand harus jelas
- informasi penting mudah diekstrak AI

### E. Guardrails Maintenance
- maintenance manual harus tetap realistis
- update halaman baru tidak boleh merusak struktur lama
- perubahan OG, favicon, metadata, schema, dan tracking harus punya checklist
